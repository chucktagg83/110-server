@ --> a Decorator

@app.route("/user", methods=["GET"])
def get user():
	return {"name": "John"}

1> Decorator (function wrapper)
2> Path (URL ex. "/home")
3> Method (GET/PULL etc..)
4> Function (def home():)
5> Return ({message: ""})
